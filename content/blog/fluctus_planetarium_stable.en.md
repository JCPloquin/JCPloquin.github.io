# 🎵 Fluctus Planetarium: The Stable Version Is Coming!

## Current Project Status

**Fluctus Planetarium** is entering its final development phase with a **stable version scheduled for September 2025**. We are therefore in the critical period where this new version should see the light very soon.

## Final Steps Before Release

### Technical Finalization
- **Event cards balancing**: Completed last week
- **Rules proofreading**: In progress, version 1.14.18
- **Printed prototypes**: Finalized, "blind play" tests were conducted in August

### Testing Phase
The game adaptation has **entered the polishing phase**, a crucial step that determines the final quality of the user experience. Blind tests will validate the game's ergonomics and accessibility.

## Innovation: From Installation to Interactive Game

### Original Concept
Fluctus Planetarium was born in 2023 from an artistic collaboration between **Louis Jeanne** and myself. Initially conceived as an installation for 8 giant screens with a 5.1 electroacoustic system, the work explores the encounter between abstract visual projections and spatialized musical compositions.

### Game Transformation
Faced with the logistical and financial challenges of contemporary art installations, the project transformed into an **interactive gaming experience** accessible to the general public, thus democratizing access to this artistic creation.

## Revolutionary Game Mechanics

### Reactive Audio System
The major innovation lies in the **reactive audio system**: the closer a meteorite gets to a planet, the more audible that world's specific musical composition becomes, transforming the sound space into a musical exploration environment.

### Gravitational Physics
The gameplay draws directly from **celestial physics**. Meteorite trajectories don't follow straight lines but curve under the gravitational effect of planets, creating controlled unpredictability that forms the core of the experience.

## Collaborative Creation Process

### The Creation Workshop
The stable version will offer a **complete creative pipeline** allowing players to create their own planet:

1. Import a portrait (landscape format required)
2. Deconstruction into abstract texture
3. Texture sonification
4. Acousmatic retouching
5. Integration into the game universe
6. Application to a planetary sphere
7. Physical parameters (mass, gravitation, rotation)

## Transmedia Perspectives

This stable version fits into my **transmedia approach**, where literature, electroacoustic composition, and game creation converge toward a coherent creative ecosystem.

***

The imminent release of this stable version represents the culmination of an ambitious artistic approach: transforming a contemporary art installation into an accessible creation tool, where each player becomes a co-creator of the sound and visual universe imagined by the artists.
